Manual MilestonePSTools installation

Run these commands in PowerShell on the PC running this app.

Install-PackageProvider -Name NuGet -Scope CurrentUser -Force
Set-PSRepository -Name PSGallery -InstallationPolicy Trusted
Install-Module -Name MilestonePSTools -Scope CurrentUser -Force -AllowClobber

If PowerShell asks to install NuGet, answer Yes. If it asks whether to trust PSGallery, answer Yes or Yes to All. After installation, run Get-Command Export-VmsHardware and Get-Command Import-VmsHardware.
